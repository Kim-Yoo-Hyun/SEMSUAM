# Paper Card

## Problem

### 사실

Unknown and unstructured environment에서 robot exploration은 map coverage뿐 아니라 SLAM subsystem의 pose/map quality에 의해 영향을 받는다.

### 논문 주장

기존 information-theoretic exploration은 localization error가 이미 해결되어 있다고 가정하는 경우가 많고, 이 가정은 practical robot deployment에서 path planning failure나 semantic map label error로 이어질 수 있다고 주장한다.

### 에이전트 추론

이 논문은 `CAND-01` Step 4-5의 직접 근거다. Semantic information gain과 SLAM pose-graph uncertainty를 같은 action utility에 넣고, navigation/exploration action이 map/pose metric을 바꾸는지 평가한다.

## Core Idea

Semantic mutual information과 pose graph spectral connectivity를 결합한 utility로 candidate exploration trajectory를 평가한다. 핵심은 semantic map uncertainty만 줄이는 것이 아니라, pose graph topology를 통해 localization uncertainty와 loop closure opportunity까지 action selection에 반영하는 것이다.

## Input / Output

Input:

- RGB-D observation
- Habitat simulator semantic segmentation
- semantic octomap
- ORB-SLAM2 / g2o 기반 pose graph
- frontier candidate paths generated by A*
- path cost and candidate trajectory time horizon

Output:

- selected exploration trajectory
- updated metric-semantic map
- updated pose graph
- exploration and SLAM evaluation traces

## Method

- multi-class occupancy grid에서 semantic mutual information을 계산한다.
- Semantic mutual information은 frontier endpoint만이 아니라 candidate path를 따라 누적한다.
- Pose graph의 Fisher Information Matrix를 직접 계산하지 않고, weighted Laplacian graph의 D-optimality로 근사한다.
- Loop closure 가능성이 있는 edge에 더 큰 weight를 두어 localization uncertainty reduction을 유도한다.
- Shannon-Rényi entropy 형태의 utility로 semantic information gain, pose graph optimality, travel cost를 균형화한다.
- 3D map을 2D로 project해 frontier를 찾고, A*로 frontier center까지 path를 만든 뒤 utility로 trajectory를 선택한다.

## Main Claims

### 논문 주장

- Habitat / Matterport3D 기반 photorealistic simulation에서 exploration efficiency를 state-of-the-art exploration methods와 비슷하게 유지한다고 주장한다.
- Metric-semantic SLAM 성능에서 average map error를 최대 21% 줄이고, semantic classification accuracy를 약 9% 개선한다고 주장한다.
- Average localization error는 최대 38% 줄었다고 보고한다.
- Information-theoretic semantic exploration과 pose graph optimality를 결합하면 SSMI, TARE보다 semantic IoU와 SLAM map/pose error가 좋아진다고 주장한다.

## Strengths

- SLAM state quality를 exploration objective에 직접 넣는다.
- `ATE`, map error, semantic `IoU`, explored volume을 함께 본다.
- Semantic uncertainty와 pose graph uncertainty를 single utility에서 다룬다.
- Code가 공개되어 있어 Step 4-5 baseline 후보로 검토 가능하다.
- `SSMI`와 `TARE`를 비교 대상으로 사용해 semantic mapping과 exploration 양쪽 baseline을 가진다.

## Limitations

- Simulation 중심 평가라 real robot transfer는 별도 검증이 필요하다.
- Habitat simulator의 semantic segmentation을 사용하므로 real-world detector noise와 open-vocabulary label uncertainty는 제한적으로만 반영된다.
- Pre-explored semantic memory를 재사용하는 문제까지는 다루지 않는다.
- ObjectNav, wrong-goal visit, wasted path 같은 task behavior metric은 없다.
- 논문 결과는 two Matterport3D environments와 repeated trials 중심이며, standard ObjectNav split은 아니다.
- Baseline 일부에서 localization loss가 발생한 trial을 fair comparison을 위해 제외했다고 설명한다. 이 처리 방식은 failure-rate metric을 따로 보지 않으면 deployment risk를 낮게 볼 수 있다.
- Pose graph가 커질 때 weighted Laplacian 계산 overhead가 커질 수 있다.

## Relevance to My Research

### 사실

`CAND-01`은 Step 4-5에서 SLAM uncertainty proxy와 map/pose consistency metric을 필요로 한다.

### 에이전트 추론

이 논문은 `CAND-01`의 Step 4-5를 설계할 때 가장 직접적인 active SLAM utility reference다. 특히 weighted Laplacian D-optimality, loop closure-aware edge weighting, `ATE`, map error, semantic `IoU`는 `SemanticSLAM` utility와 evaluation contract의 후보가 된다.
