# docs

## Role

`docs/`는 repo-level workflow와 판단 기준을 관리한다.

## Reading Order

1. `index.md`
2. `paper.md`
3. `reproducibility.md`
4. 작업에 따라 `literature.md`, `hypothesis.md`, `experiments.md`

## Rules

- 세부 실험 로그를 길게 저장하지 않는다.
- 논문 novelty와 reviewer-defense 판단은 `paper.md`를 우선한다.
- data, checkpoint, Docker, artifact 판단은 `reproducibility.md`를 우선한다.
